# 🔹 What is `ObjectMapper` really?

`ObjectMapper` is a **data binding engine**, not just a JSON parser.

It can convert between:

1. JSON String ↔ Java Object
2. Java Object ↔ JSON String
3. **Java Object ↔ Java Object** ✅
4. Map ↔ Java Object

---

# 1️⃣ Traditional Use (Most People Know This)

### JSON String → Java Object

```java
User user = objectMapper.readValue(jsonString, User.class);
```

### Java Object → JSON String

```java
String json = objectMapper.writeValueAsString(user);
```

✔ This is **serialization / deserialization**

---

# 2️⃣ Java Object → Java Object Conversion (LESS KNOWN, VERY IMPORTANT)

### Yes, ObjectMapper supports this using `convertValue()`

---

## 🔹 Example: Entity → DTO conversion

### Entity

```java
class UserEntity {
    private Long id;
    private String name;
    private String email;
    private String password;
}
```

### DTO

```java
class UserResponseDTO {
    private Long id;
    private String name;
    private String email;
}
```

---

### ✅ Conversion using ObjectMapper

```java
UserResponseDTO dto = objectMapper.convertValue(userEntity, UserResponseDTO.class);
```

✔ No JSON string involved
✔ Field mapping by name
✔ Extra fields ignored automatically

---

# 3️⃣ Common Production Use Cases

## ✔ Entity → DTO

```java
UserResponseDTO dto = objectMapper.convertValue(entity, UserResponseDTO.class);
```

## ✔ DTO → Entity

```java
UserEntity entity = objectMapper.convertValue(dto, UserEntity.class);
```

## ✔ Map → Object

```java
User user = objectMapper.convertValue(map, User.class);
```

## ✔ Object → Map

```java
Map<String, Object> map = objectMapper.convertValue(user, Map.class);
```

---

# 5️⃣ ObjectMapper vs Manual Mapping vs MapStruct

| Approach       | When to Use                 |
| -------------- | --------------------------- |
| Manual setters | Complex logic               |
| ObjectMapper   | Simple DTO mapping          |
| MapStruct      | Large projects, performance |
| ModelMapper    | Dynamic but slower          |

📌 **Enterprise best practice**:

* Small apps → ObjectMapper
* Large apps → MapStruct
